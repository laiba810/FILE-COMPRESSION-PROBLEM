PROJECT: File Compression Using Huffman Coding

Language:
Java

Data Structures Used:
1. HashMap
2. Priority Queue (Min Heap)
3. Binary Tree

Algorithm:
Huffman Coding (Greedy Algorithm)

- Build Huffman Tree
- Generate Huffman Codes
- Compress Text
- Decompress Text
- Calculate Compression Ratio

Run:
1. Compile all Java files
2. Run Main.java
3. Enter text
4. View results
